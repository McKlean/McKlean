## Who Am I?

I'm **McKlean**, co-developer behind **TinySprite Scripts**.

I focus on building and maintaining content across multiple communities, with experience in:

- **FiveM Scripting** — creating and supporting custom server resources  
- **FiveM Mapping Support** — resolving **collisions, occlusions, and MLO conflicts** *(no 3D modelling or Blender work)*  
- **Discord Bot Development** — building custom bots and automation systems  
- **Amateur Hytale Scripting**  
- **Amateur Minecraft Scripting**  
- **Amateur Program Development**

I enjoy solving problems, improving server performance, and helping communities run smoothly through reliable tools and systems.
